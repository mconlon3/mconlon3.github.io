Learning HTML

What is HTML?

HTML is the foundation for websites... like the one you're viewing right now!

HTML stands for HyperText Markup Language.

The HTML filetype is made up of HTML code, which is made up of elements.

Basic Structure

To begin with HTML, you'll need to know the basics.

  • <!DOCTYPE html> indicates that the language is HTML.
  • <head> indicates important data about the website that isn't shown visually. For example, it includes the title of the webpage.
  • <body> is the main part of your website, what will be shown to the user.
  • All three of these elements must include closing tags, which will look like this:
  • <head></head>
  • It's important to remember this!

Text Elements

These elements will help you display text on your page.

  • <h#>: Heading elements, replace the # with a number 1 through 6, 1 being the most important and 6 being the least.
  • <p>: Paragraph element, used to show text.
  • <a>: Anchor element, used for hyperlinks.
  • <br>: Line break element, self explanatory.
  • <blockquote>: An element that appears as a quote.

List Elements

These elements will help you display lists. If you notice the bullet points below, that's an unordered list!

  • <ul>: Unordered list, typically displayed with bullet points.
  • <ol>: Ordered list, typically displayed with numbers.
  • <li>: List item used inside <ul> and <ol>.

Image and Media

These elements will allow you to add visuals, preventing your website from just being filled with text.

  • <img>: Image tag, used to embed images into the webpage. The src attribute is used to specify the image location.
  • <audio>: Embeds audio files, allowing users to play them on your page.
  • <video>: Embeds video files on the page.

Table Elements

These elements will show a table.

  • <table>: Defines the table.
  • <tr>: Table row.
  • <th>: Table header cell.
  • <td>: Table data cell.

Replace elementName with the element you are modifying, property with the property you will modify, and value with the value.
