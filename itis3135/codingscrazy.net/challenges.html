Fun Challenges

Feeling confident? Or just looking for something harder?

Try out these challenges!

Here are five projects you can try, as well as the skills they will test and reasons why it is a challenge.

  1. Edit a Wikipedia page.

    2. HTML doesn't mean you have to start from scratch!

    Since websites are built using HTML, that also means you can edit them.

    Try editing a page of Wikipedia (don't add any misinformation!) using HTML.

  2. Make a website about yourself.

    3. Websites can be an easy way to introduce yourself.

    You can list a lot of information at once using lists.

    Use a stylesheet to make the website look nice.

  3. Create a fake product page.

    4. Incorporate images onto your website.

    Add multiple pages if you think you can, as well as a way to navigate the website.

    An embedded video could add extra realism.

  4. Music Player

    5. Try to create a music player.

    This project will test your ability to incorporate clickable objects.

    You will also have to create a file structure to store the audio files. (Unless you stream them another way.)

  5. Portfolio

    6. Show off all your HTML projects in a portfolio.

    Make sure to keep the website interesting instead of just a list of projects.

    Try adding images, slideshows, and clickable objects!
